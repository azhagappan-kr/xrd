Practical Guidance

Step-by-step approaches for common analysis tasks.

üîç Phase Identification Workflow

  1. Data quality check: Examine the raw pattern for noise, peak shapes, and background. Apply background subtraction if necessary.
  2. Peak detection: Use the software's automatic peak-finding function, then manually verify each peak position. Record d-spacings from 2Œ∏ values using Bragg's law.
  3. Search-match: Run an automated database search. Evaluate the candidate matches by comparing the top-scoring hits with your pattern, focusing on the strongest (non-overlapping) peaks.
  4. Validate the match: Confirm that all major peaks in your pattern are accounted for by the identified phase(s). Check that relative peak intensities are broadly consistent.
  5. Multi-phase samples: After identifying the dominant phase, subtract its contribution or focus on unidentified residual peaks and repeat the search for secondary phases.
  6. Quantitative phase analysis: Use Rietveld refinement (GSAS-II, FullProf) to determine weight fractions of each phase in a mixture.
